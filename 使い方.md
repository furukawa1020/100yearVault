## 使い方

### 起動方法

`vaultapp` フォルダで以下のコマンドを実行してください。

```bash
go run .
```

または、ビルド済みの実行ファイル（`vaultapp.exe` / `vaultapp`）を直接起動してください。

---

### 画面の説明

#### 1. メインスクリーン（記憶の星雲）

アプリを起動すると、保存された記憶が光の粒として星雲状に表示されます。

- **記憶の選択**: 画面中央に浮かぶ粒（記憶）にマウスを近づけると強調表示されます。クリックすると記憶の詳細画面へ移動します。
- **新規記憶の作成**: 画面中央の「特異点（シンギュラリティ）」にマウスをホバーしてクリックすると、記憶の作成画面へ移動します。
- **ウェブカメラ連動**: カメラが接続されている場合、顔の位置・距離に応じて星雲が反応します。

#### 2. 記憶の作成画面（コンポーズ）

新しい記憶（暗号化テキスト）を保存します。

| 項目 | 説明 |
|------|------|
| **記憶の題名** | 記憶のタイトルを入力します（必須） |
| **記憶の内容** | 保存したいテキストを入力します |
| **合言葉** | 設定すると記憶の閲覧時にパスフレーズが必要になります（省略可） |
| **封印する** | 内容を暗号化して保存します |
| **戻る** | メインスクリーンへ戻ります |

> **注意**: 合言葉を設定した場合、忘れると記憶を復元できません。大切に保管してください。

#### 3. 記憶の儀式画面（リチュアル）

選択した記憶の内容を閲覧・追記します。

| ボタン | 説明 |
|--------|------|
| **同調する（アンロック）** | 合言葉を入力して記憶を復号・表示します |
| **地層を追加** | 既存の記憶に新しい考察（レイヤー）を重ねて保存します |
| **戻る** | メインスクリーンへ戻ります |

---

### 記憶のライフサイクル

1. **作成**: タイトルと内容を入力し、必要に応じて合言葉を設定して封印します。
2. **閲覧**: メイン画面から記憶を選択し、合言葉を入力して内容を確認します。
3. **追記（地層）**: 時間が経ったあとに感想や振り返りを「地層」として追加できます。地層は時系列で記録されます。

---

### データの保存場所

| ファイル / フォルダ | 内容 |
|---------------------|------|
| `vault.db` | 記憶のメタデータ（タイトル・日時など） |
| `vaults/` | 暗号化された記憶の本文（`.age` ファイル） |
