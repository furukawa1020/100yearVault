# 百年のヴォールト（Hundred-Year Vault）

**百年先の自分へ、今の想いを封印する。**

時を超えた記憶の保管庫。暗号化された「記憶の断片」を保存し、将来の自分（あるいは誰か）に届けるための、デスクトップアプリケーションです。

---

## 概要

百年のヴォールトは、大切な想いや記録を暗号化して保存し、未来へ手渡すためのツールです。ウェブカメラによる顔認識インタラクション、神経細胞をイメージしたアニメーションUI、そして堅牢な暗号化技術によって、ただのメモ帳とは一線を画した体験を提供します。

---

## 主な機能

- **記憶の封印**: タイトルと本文をパスフレーズで暗号化し、「記憶の断片」として保存します。
- **地層（レイヤー）システム**: 既存の記憶に「振り返り」を重ねて追記できます。地質の地層のように、時間の積み重ねを可視化します。
- **儀式による解錠**: 保存した記憶を専用の「儀式」画面でパスフレーズを用いて復号します。
- **ウェブカメラ連動UI**: カメラで顔を認識し、視線や距離に応じてUIがリアクションします。カメラがない場合はデモモード（ゴーストアバター）で自動的に動作します。
- **ニューラルアニメーション**: 神経ネットワークを模した動的な粒子アニメーションがバックグラウンドで常時稼働します。
- **強固な暗号化**: [age](https://github.com/FiloSottile/age) ライブラリによるモダンな暗号化を採用しています。

## 使い方

詳細な画面説明や操作手順は [使い方.md](./使い方.md) を参照してください。

### 記憶を封印する

1. メイン画面（ニューラルネットワーク画面）の中央にある「特異点」をクリックします。
2. 「記憶の封印（Compose）」画面が開きます。
3. タイトル、本文、任意でパスフレーズを入力し、「封印」ボタンを押します。

### 記憶を解錠する（儀式）

1. メイン画面に浮かぶ記憶の断片をクリックします。
2. 「儀式（Ritual）」画面が開きます。
3. パスフレーズを入力し、「解錠」ボタンを押すと、数秒の処理後に内容が表示されます。

### 地層を追加する

1. 儀式画面で記憶を開いた後、「地層を追加」ボタンをクリックします。
2. 振り返りの内容を入力して封印することで、元の記憶に新しい層が重なります。

## ビルド・実行方法

### 必要な環境

- **Go 1.24 以上**
- **CGO が有効**（SQLite のビルドに必要）
  - Windows: [TDM-GCC](https://jmeubank.github.io/tdm-gcc/) などの C コンパイラが必要です。
  - macOS: Xcode Command Line Tools（`xcode-select --install`）
  - Linux: `gcc` および `libgl-dev` 等のグラフィックライブラリ

### 実行

```bash
cd vaultapp
go mod download
go run .
```

### ビルド（実行ファイルの生成）

```bash
cd vaultapp
go build -o vaultapp .
```

> **注意**: ウェブカメラ連動機能（顔認識）を利用する場合、`assets/models/` ディレクトリに `facefinder.bin` と `puploc.bin` が必要です。これらのファイルが存在しない場合、アプリはデモモード（ゴーストアバター）で自動起動します。

## ディレクトリ構成

```
100yearVault/
├── README.md            # このファイル
├── 使い方.md             # 詳細な操作ガイド
└── vaultapp/
    ├── main.go          # エントリーポイント・メインループ・ウェブカメラ処理
    ├── internal/
    │   ├── crypto/      # age 暗号化 / 復号ロジック
    │   ├── db/          # SQLite 操作（記憶・地層の保存・読み込み）
    │   ├── ui/          # Gio UI コンポーネント・アニメーション
    │   └── vault/       # データ構造定義（MemoryFragment, Layer 等）
    ├── assets/
    │   ├── fonts/       # フォントファイル
    │   └── models/      # pigo 顔検出モデル（facefinder.bin, puploc.bin）
    └── vaults/          # 暗号化された記憶ファイル（.age）
```

## データの保存場所

| ファイル / フォルダ | 内容 |
|---------------------|------|
| `vaultapp/vault.db` | 記憶のメタデータ（タイトル・日時など） |
| `vaultapp/vaults/`  | 暗号化された記憶の本文（`.age` ファイル） |

> **注意**: `合言葉（パスフレーズ）`を設定した場合、忘れると記憶を復元できません。大切に保管してください。
