 (Hundred-Year Vault)

**百年先の自分へ、今の想いを封印する。**

時を超えた記憶の保管庫。暗号化された「記憶の断片」を保存し、設定した日数が経過した後に解錠できる、デスクトップアプリケーションです。

---

## 概要

百年のヴォールトは、大切な想いや記録を暗号化して保存し、将来の自分（あるいは誰か）に届けるためのツールです。ウェブカメラによる顔認識インタラクション、神経細胞をイメージしたアニメーションUI、そして堅牢な暗号化技術によって、ただのメモ帳とは一線を画した体験を提供します。

---

## 主な機能

- **記憶の封印**: タイトルと本文をパスフレーズで暗号化し、「記憶の断片」として保存します。
- **地層（レイヤー）システム**: 既存の記憶に「振り返り」を重ねて追記できます。地質の地層のように、時間の積み重ねを可視化します。
- **儀式による解錠**: 保存した記憶を専用の「儀式」画面でパスフレーズを用いて復号します。
- **ウェブカメラ連動UI**: カメラで顔を認識し、視線や距離に応じてUIがリアクションします。カメラがない場合はデモモード（ゴーストアバター）で動作します。
- **ニューラルアニメーション**: 神経ネットワークを模した動的な粒子アニメーションがバックグラウンドで常時稼働します。
- **強固な暗号化**: [age](https://github.com/FiloSottile/age) ライブラリによるモダンな暗号化を採用しています。

## 使い方

### 記憶を封印する

1. メイン画面（ニューラルネットワーク画面）の中央にある「特異点」をクリックします。
2. 「記憶の封印（Compose）」画面が開きます。
3. タイトル、本文、任意でパスフレーズを入力し、「封印」ボタンを押します。

### 記憶を解錠する（儀式）

1. メイン画面に浮かぶ記憶の断片をクリックします。
2. 「儀式（Ritual）」画面が開きます。
3. パスフレーズを入力し、「解錠」ボタンを押すと、数秒の処理後に内容が表示されます。

### 地層を追加する

1. 儀式画面で記憶を開いた後、「地層を追加」ボタンをクリックします。
2. 振り返りの内容を入力して封印することで、元の記憶に新しい層が重なります。

## ビルド方法

```bash
# 依存関係の取得
cd vaultapp
go mod download

# 実行
go run .
```

> **注意**: ウェブカメラ連動機能を利用する場合、`assets/models/` ディレクトリに `facefinder.bin` と `puploc.bin` が必要です。

---

## ディレクトリ構成

```
vaultapp/
├── main.go              # エントリーポイント・メインループ
├── internal/
│   ├── crypto/          # age暗号化/復号ロジック
│   ├── db/              # SQLite操作 (記憶・地層の保存/読み込み)
│   ├── ui/              # Gio UIコンポーネント・アニメーション
│   └── vault/           # データ構造定義 (MemoryFragment, Layer等)
├── assets/
│   ├── fonts/           # フォントファイル
│   └── models/          # pigo顔検出モデル
└── vaults/              # 暗号化された記憶ファイル (.age)
```
